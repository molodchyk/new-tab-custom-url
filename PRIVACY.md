# Privacy Policy

New Tab: Custom URL stores only the settings needed to load the page chosen by
the user.

Stored settings may include:

- the configured new-tab URL
- focus mode preference
- extension theme preference
- whether Chrome sync storage should be used

The extension does not collect analytics, does not sell data, does not show ads,
and does not make remote network requests.

Browser permissions:

- `storage`, used to save the settings listed above in Chrome extension storage
- `file:///*`, used only so Chrome can expose local-file access for
  user-selected `file://` new-tab pages

The extension requests file URL access only so Chrome can open a user-selected
local file as the new-tab page. It does not read file contents or upload local
file paths.

When sync storage is enabled, Chrome may sync the extension settings through the
user's browser profile according to the user's Chrome sync configuration.
