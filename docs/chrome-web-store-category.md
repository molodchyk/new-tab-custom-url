# Chrome Web Store Category Decision

Selected category: Functionality and UI

Reason:
New Tab: Custom URL changes the browser's new-tab behavior and gives users a
small interface for configuring that behavior. It is primarily a browser UI and
functionality customization extension, not a productivity workflow product,
developer tool, privacy/security product, or general-purpose utility.

## Notes

Tools would also be defensible because similar redirect extensions sometimes use
that category, but Functionality and UI is more precise for an extension whose
main purpose is replacing Chrome's default new-tab page with a user-selected
page.
